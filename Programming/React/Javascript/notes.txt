//Básico de Javascript
-Variáveis: onde armazenamos e manipulamos, como atribuições ou concatenações
-As variáveis são case sensitive
-Ela tem palavras reservadas
-Comentários: usar // ou /**/
-Pomemos usar eventos para deixar nosso site interativo
-Funções: organiza e evita de ter que repetir várias vezes o mesmo código
-Array: Lista, muito útil e muito utilizado no mercado de trabalho
-Temos métodos de arrays para manipulá-los como: join = junta e sepera por algo, indexOf = sabe o índice, pop = Remove o último, shift = remove o primeiro
-Podemos alterar sabendo a posição e atribuindo um novo valor

//Evoluindo no javascript
-Loops: podemos usar while, for, do while
-Switch: podemos usar para evitar vários if else
-Temporizador: passar um eventos em determinado tempo. Usar setInterval() para infinitamente, ou setTimeout para exercutar uma vez só
-WebStorage: Salvar alguma informação do usuário. setItem, getItem, removeItem
-Podemos usar a mesma lógica do webstorage para os sessionStorage
-Let, var, const: var sai do escopo enquanto let e const não. Const não deixa alterar o valor e deve ser a preferência num desenvolvimento, alterando somente quando necessário para let.
-Template literals: Usar o `` para concatenar strings mais fáceis. Utilizar o ${nomeDaVariavel} para concatenar variáveis
-Descontrução em Javascript: Desestruturamos objetos e arrays
-Spread operator: Podemos juntar as operações, consequentemente usando menos linhas
-rest operator: Usado quando não sabemos quantas entradas terá em nossa função, ele repete para cada entrada
-Função anônima: podemos usar uma função sem nome ou usar arrow function sem as {} como parâmetro
