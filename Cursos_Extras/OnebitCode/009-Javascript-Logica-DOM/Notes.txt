//O que são os eventos
-Forma dinâmica de executar javascript
-Executadas se o usuário fazer ações específicas na página 

//Eventos
-onmouseenter
-onchange
-onfocus
-onblur	
-onclick
-como parâmetro, podemos criar uma nova função em um arquivo js externo e usar nos eventos

//DOM
-Modelo de objeto de documento
-Modelo da nossa página do documento em objeto
-Pode ler, criar, excluir, modificar elementos, modificar estilo

//Obtendo elementos HTML do DOM
-Obter vários elementos HTMLColletion: getElementsByTagName(), getElementByClassName()
-Obter vários elementos NodeList: getElementsByName(), querySelectorAll()
-obter um único elemento: getElementById(), querySelector()

//Criando novos elementos no DOM
-document.createElemente()
-.innerHTML adiciona conteúdo ao elemento
-.appendChild(newElement) adiciona um novo nó
