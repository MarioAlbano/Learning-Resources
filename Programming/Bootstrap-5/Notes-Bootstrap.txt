-----Bootstrap------

//O que é?
-O bootstrap é uma biblioteca de componentes, em que partes já vem criadas e cabe a nós alterá-las
-Agiliza o processo de criação e tem um sistema de grid muito de fácil adaptação ao responsivo

//Instalação
-CDN, Arquivos internos do nosso servidor(não recomendado), npm

//Como utilizar o bootstrap?
-Após adicionar o link de css e se necessário do js, olhar documentação e adicionar elemento. Após isto, conferir na inspecionar qual o atributo que deseja alterar e fazer a modificação em um arquivo externo de css

//Breakpoints
-Pontos específicos de resolução criados por media query para deixar o site responsivo
