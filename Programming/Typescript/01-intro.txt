//Typescript
-Muito utilizado pelo mercado
-Acompanhado de frameworks populares
-Evita muitos bugs em grandes aplicações e facilita a documentação e organização do código
-Não é uma linguagem de programação
-É um superset do javascript
-Javascript é uma linguagem fracamente tipada
-Depois do desenvolvimento ele é convertido para javascript novamente

//Instalando o typescript
-Entrar no site oficial do typescript
-rodar no terminal "npm install -g typescript"
-para testar se funcionou usar no terminal "tsc --version"

//Utilizando o typescript
-criar um arquivo .ts na pasta
-clicar bom o botao direito na pasta usando o visual code e clicar em abrir terminal
-depois compilar abrindo no terminal e digitando "tsc nomeDoSeuArquivoTs"; 
-o typescript deixa o codigo mais enxuto
