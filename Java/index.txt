//Instalação
-JDK e IDE (IntelliJ)

//Como funciona
-IDE passa o código para o JDK, Joga para o JRE(compilador) que usa uma máquina virtual(JVM) e passa para o computador

//Edições
-Java SE = Java normal
-Java EE = Java para empresas
-Java ME = Java para celulares
-Java FX = Java para desktops

//Organização
-Package
-Class
-Method
-Statements

//Variáveis
-Primitive types: (byte(128), short(32k), int(2b), long, float, double, char, boolean). Não podemos usar os métodos delas
-Reference Data Types: Armanezam referência de objetos. Podemos usar métodos
