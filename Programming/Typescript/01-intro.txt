//Typescript
-Linguagem de programação com base no JavaScript, adiciona tipagem estática ao Javascript
-Todo código Typescript é convertido para js pelo Typescript Compiler
-Evita erros, deixa a IDE "mais inteligente", recursos extras como tuplas, enums, melhor suporte a POO
-Fortemente tipada. Tipagem dinâmica e tipagem estática
-Liguagens estáticas que possuem compiladores
-Vantajoso em projetos grandes

//Criando código Typescript
-No terminal, usar "tsc index.ts" para criar um arquivo js
-Configurar o typescript: Terminal e usar "tsc --init": Podemos configurar a versão, módulo, pasta de saída do arquivo JS, pasta que será procurado o arquivo ts
