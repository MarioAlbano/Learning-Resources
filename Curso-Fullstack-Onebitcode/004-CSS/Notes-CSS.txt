//CSS
-Cascade Style Sheets, folhas de cascata
-Linguagem para definir o estilo
-Não deve ser definida dentro do HTML

//Como funciona
-	Seletor {Declaração(Propriedade: valor)}
-Ex:      h1 {			color:        #fff	}
-Criamos um style externo para evitar repetição em várias páginas

//Cores CSS
-Por nome, RGB, Hexadecimais (mais utilizada), HSL, HWB

//Partes importantes:
-Background
-Border
-Seletores

//Seletores especiais
-* (todos os elementos)
-Pai para filho (um seletor e depois outro)
-Descendência (elemento > dentro dele outro elemento)
-Adjacente + (selecionando somente oque vem após)
-Seletor de atributor (a[href="/recibo"] {}), pseudo-classes)

//Especifificidade
Determina quais regras serão aplicadas

//Herança
Quando por exemplo definimos uma cor para a Div os elementos herdam isto

//Texto e fontes
-Deixar a que mais desejamos, outra que desejamos, e uma genérica

-Content
-Padding
-Border
-Margin
